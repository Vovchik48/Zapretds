# Zapret Android VPN

VPN приложение для обхода блокировок

## Функции
- ✅ Обход DPI блокировок
- ✅ Telegram, YouTube, Discord
- ✅ Без root прав
- ✅ Бесплатно

## Установка
1. Скачайте APK из [Releases](https://github.com/Vovchik48/Zapretds/releases)
2. Установите на Android устройство
3. Нажмите START VPN

## Сборка
APK собирается автоматически через GitHub Actions
